# Black Friday Sales Prediction

## Overview
This project uses Machine Learning techniques to predict customer purchase amounts during Black Friday sales. The objective is to analyze customer demographics and product-related features to forecast purchase behavior accurately.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost (if used)
- Matplotlib
- Seaborn

## Dataset
The dataset contains customer and transaction information, including:

- User ID
- Product ID
- Gender
- Age
- Occupation
- City Category
- Years in Current City
- Marital Status
- Product Categories
- Purchase Amount

## Project Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Sales Prediction
- Model Evaluation

## Installation

1. Clone the repository

```bash
git clone https://github.com/pennadasahitya2003-wq/black-friday-sales-prediction.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

## Running the Project

Open the Jupyter Notebook and run all cells:

```bash
jupyter notebook
```

## Results

The machine learning model predicts customer purchase amounts based on demographic and product-related attributes. The project demonstrates the use of feature engineering and regression techniques for sales forecasting.
